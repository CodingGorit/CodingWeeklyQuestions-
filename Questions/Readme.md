# 第一周
### 斐波那契数列
#### 1. 问题描述
> 题目：古典问题：有一对兔子，从出生后第3个月起每个月都生一对兔子，小兔子长到第三个月后每个月又生一对兔子，假如兔子都不死，问每个月的兔子对数为多少？ 

#### 2. 问题分析
只有输出，没有输入，比如：1,1,2,3,5....

#### 3. 小提示
1. 可以使用迭代，也可以使用递归的方式
2. 斐波那契数列，一个数等于它前两项之和，如 a[i] = a[i-1] + a[i-2]
